# Syntax (non-normative notes)

> **This document is not the normative syntax reference.**
> The normative grammar is [`docs/language/grammar.md`](language/grammar.md),
> generated from [`crates/reportage-core/src/reportage.pest`](../crates/reportage-core/src/reportage.pest).

The content that was here described syntax constructs that are not yet
declared in `reportage.pest` (e.g. `before_each`, `params`, `variant`,
`file` heredocs). Those constructs are not yet in the grammar.

- For the current v0 syntax, see the generated [`docs/language/grammar.md`](language/grammar.md).
- Constructs not in `reportage.pest` are not part of v0 and are out of scope
  for the normative grammar document.
